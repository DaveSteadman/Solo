Solo

Start SoloHub from this repository root:

```
python .\SoloHub\code\main.py
```

Then open:

http://127.0.0.1:9700/

Press Ctrl+C in the terminal to stop it.

SoloHub starts services marked `autoStart` in `Config/hub.json`. Service ports and data paths are read from `Config/factory-default.json`, with local overrides in `Config/local.json`.
